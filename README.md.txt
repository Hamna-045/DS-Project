                      Data Science Project
	        COVID-19 Dataset Visualizations

The Project consists of Visualizations on the COVID'19 data set, mainly of March'2020 to May'2020.


The tool used to create these visualizations is Tableau Public. It is an easy to use tool available for free, for making compelling and informative visualizations. This tool also provides data cleaning facility on just single button click without writing any code for creating visuals or for cleaning data.
